# brackets.io 에서프로그램 설치
![브라켓설치 바로가기](http://brackets.io)
> 프로그램 설치시 필요한 plugin

* emmet : 코드단축 기능
* custom tabs : html/css아이콘모양
* cssfier : css확인관리
* markdown preview : markdown파일확인기능
* indent guide : 라인체크기능
* beautify : 텍스트 정렬체크
* cdn : cdn파일체크
* color highlight : css색상확인
* w3validation | interactive inter : 문법오류검사
* swatcher : 색상확인
